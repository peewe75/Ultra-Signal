# SOFTIBRIDGE PROMPT: LANDING PAGE (VETRINA & VENDITA)

Crea una Landing Page ad alta conversione per il software di automazione trading SoftiBridge.

## 1. Visual Design

- **Tema**: Dark Mode futuristica.
- **Colori**: Nero profondo, Blu Elettrico (`#0070f3`), Bianco ghiaccio per il testo.
- **Stile**: Glassmorphism, gradienti morbidi, animazioni all'ingresso delle sezioni.

## 2. Sezioni Richieste

1. **Navbar**: Logo, Link rapidi (Features, Prezzi, FAQ), Bottone "Accedi" e "Inizia Ora".
2. **Hero Section**:
   - Titolo: "Domina i Mercati con l'Automazione Intelligente di SoftiBridge".
   - Sottotitolo: "La tecnologia bridge definitiva per MT4 e MT5. Gestisci licenze, account e segnali con un'unica dashboard professionale."
   - CTA Primary: "Ottieni la tua Licenza" (Porta alla registrazione/pricing).
3. **Features Grid**:
   - "Multi-Terminal": Gestisci più account simultaneamente.
   - "Sicurezza Totale": Licenze criptate e controllo VPS avanzato.
   - "Zero Latenza": Sincronizzazione immediata dei segnali.
4. **Pricing Table**:
   - Tre schede: BASIC, PRO, ENTERPRISE.
   - Evidenzia il piano PRO come "Più Scelto".
   - Mostra chiaramente i limiti (1, 3 o 10 account autorizzati).
5. **Footer**: Link legali, contatti supporto, social.

## 3. UX & Functional

- Utilizza `Layout.js` per una struttura coerente.
- Ottimizzazione SEO: Meta tags per "Trading Automation", "MT4/MT5 Bridge".
- Responsive design perfetto per mobile e tablet.
